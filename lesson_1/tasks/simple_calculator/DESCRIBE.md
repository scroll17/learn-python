# Simple calculator

**You must realize logic which is equal to sample below.**

**Supported operators:**
- `+`
- `-`
- `*`
- `/`

## 1. Expected behavior

```text
>>> input init number: 0
>>> action: + 5
>>> state => 5
>>> action: - 2
>>> state => 3
>>> action: * 3
>>> state => 9
>>> action: / 9
>>> state => 1
>>> action: exit
>>> calculator terminated....
```
